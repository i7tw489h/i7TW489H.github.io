2.4training

在readme中存在这几行说明

  • Set parameters from .config.ymal
  • Set model from litmodel.py
  • Run training.py

Set parameters from .config.ymal的意思是：先配置训练参数（数据集路径、训练轮数、批次大小、GPU 编号等），这些参数都写在 ADFCNN-MI/ADFCNN/configs/ 下的 yaml 文件里（比如bcicompet2a_config.yaml、bcicompet2b_config.yaml）。

实操：打开对应任务的 yaml 文件（比如做 BCI-IV 2a 任务就改bcicompet2a_config.yaml）；

重点修改以下参数：

  • BASE_PATH：数据集存放路径（比如 BCI-IV 2a 的数据集要放在./Datasets/BCI_Competition_IV/BCI_Competition_IV_2a）；
  • GPU_NUM：指定训练用的 GPU 编号（比如用 0 号卡就改"0"）；
  • num_subjects：数据集的被试数量（2a/2b 是 9，KU_MI 是 54）；
  • 其他参数（EPOCHS训练轮数、batch_size批次大小）可保持默认，也可按需调整。Set model from litmodel.py

Set model from litmodel.py：
